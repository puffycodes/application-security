# Generate Semgrep Rules for Code Scanning

## Requirement
1. Analyse the framework used in the project.
2. All the project source code must be analysed, including those in the subfolders of the project.
3. Write the generated rules to the subfolder semgrep-rules in the output folder.

## Semgrep Rules to generate
- write some semgrep rules to scan for the present of input validation in the project.
- write some semgrep rules to scan for the present of output sanitization in the project.
