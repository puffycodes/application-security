# API Endpoint Analysis

## Requirement
1. Analyse the API endpoints in details.
2. All API endpoints must be analysed and listed, including those in the subfolders of the project.
3. Write the analysis to an output file.

## Listing of API Endpoints
1. List the route for all the API endpoints in the project and all its subfolders.

## Details of API Endpoints
For each of the route, list the following:
1. The route path
1. The file where the route is implemented
1. Input parameters for the route
1. The authentication and authorization in the route
1. The input validation in the route
1. The output sanitization in the route
1. Injection Prevention mechanism
1. Rate limiting mechanism
1. Other API security measures such as CORS
