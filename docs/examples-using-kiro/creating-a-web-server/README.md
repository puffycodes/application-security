# Creating a Web Server Using Kiro

## Steps to Generate Code

Run Kiro:
```
kiro-cli
```

Specified the Features:
```
/context add templates/features.md
```

Generate the code using different framework:

Using Express and React:
```
generate the web server using express and react framework. place the source code in the folder web-server/express-react/.
```

Or Using SpringBoot:
```
generate the web server using spring-boot framework. place the source code in the folder web-server/spring-boot/.
```
