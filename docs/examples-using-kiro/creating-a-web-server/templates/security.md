# Application Security

## Code Security

Code generated should include security measures for the following.
1. Prevent SQL injection vulnerabilities in database queries.
1. Prevent Cross-Site Scripting (XSS) vulnerabilities in web endpoints.
1. Authentication and authorization mechanisms.
1. Input validation and output sanitization mechanisms.
1. No hardcoded secrets, passwords, or API keys.
1. Proper error handling that does not expose sensitive information.
1. Logging mechanism.
1. Session management and token handling.
