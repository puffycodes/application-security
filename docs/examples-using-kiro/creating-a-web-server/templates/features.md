# Features of a Web Server

## User Interface

### Functions

1. A Home page that says "Hello. Welcome to the Home Page." and a link "Submit Message" that will link to the Submission page.
1. The Submission page will ask user for 'email' and 'message' and have a 'Submit' button.
1. The 'Submit' button will send the email to the server. In return, the user will get a page showing the email and the message. The page will have a link back to the Submission page.
1. All pages, except the Home page, will have a link back to the Home page.

### Style

1. The User Interface should use a CSS Stlyesheet.
